• 전시 모듈화를 통해 페이지를 컴포넌트 단위로 조합 및 업체가 화면 구성할 수 있도록 개선

내 역할
• 백오피스 관리 기능 개발: 전시 컴포넌트 CRUD API 개발

성과
• 반복적인 개발 리소스 절감 및 전시화면 리드타임 단축
